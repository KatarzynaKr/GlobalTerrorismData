Global Terrorism Data Processing and Visualization¶

Project prepared for 'Data exploration' classes by Katarzyna Czosnyka¶

Project contains processing and visualization of Global Terrorism Dataset from Kaggle - could be found here. The quick view of database will be possible as one of the undermentioned task. In case you want to start the codes, please download the database (more insructions in 'readme') and follow the instructions below.
